# Lista de Compras

Sistema de lista de compras desenvolvido em Java no terminal, utilizando conceitos de Programação Orientada a Objetos (POO), validação de entradas e manipulação de arquivos.

---

## Funcionalidades

- Adicionar itens à lista
- Remover itens da lista
- Visualizar itens cadastrados
- Exportar lista para arquivo `.txt`
- Validação de entradas do usuário
- Tratamento de erros no terminal


---

## Conceitos Aplicados

- Programação Orientada a Objetos (POO)
- Encapsulamento
- Classes e Objetos
- Métodos
- ArrayList
- Estruturas de repetição
- Estrutura `switch`
- Validação de dados
- Tratamento de exceções
- Manipulação de arquivos

---

## Tecnologias Utilizadas

- Java
- IntelliJ IDEA
- Scanner
- ArrayList
- BufferedWriter / FileWriter
