# Lista de Compras

Sistema de lista de compras desenvolvido em Java no terminal, com funcionalidades de adicionar, visualizar e exportar itens para um arquivo `.txt`.

---

##  Funcionalidades

- Adicionar itens à lista
- Visualizar itens cadastrados
- Exportar lista para arquivo `.txt`
- Menu interativo no terminal

---

## git Tecnologias Utilizadas

- Java
- POO (Programação Orientada a Objetos)
- ArrayList
- Scanner
- Manipulação de Arquivos
